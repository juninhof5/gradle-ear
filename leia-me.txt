src/main/application         Recursos Ear, tais como um diretório META-INF


-> O plugin EAR adiciona suporte para empacotar arquivo EAR para aplicações WEB.
-> O plugin EAR adiciona a seguinte TASK(default) para o projeto
	Task name: ear
    Descrição: Monta o arquivo EAR da aplicação
-> Ele não requer o plugin Java, mas para projeto que usa o plugin java ele desabilita a geração default do arquivo JAR.
-> Para usar o plugin EAR, incluir o seguinte em seu script build: apply plugin: 'ear'
